# 任务要求
请根据提供的随访对话内容，生成新的、独特的随访对话内容。

# 随访对话内容指南
## 1. 关联性
新对话内容需与预期的随访报告结果紧密相关，确保通过新对话判断的随访报告结果与预期报告结果完全一致。

## 2. 对话内容与风格约束
1. 角色：医院工作人员（通常来自影像科或随访部门）和患者/家属，用角色1和角色2标记，角色之间互相并不熟悉。
2. 场景：发生地在中国，形式为电话随访。
3. 目的：了解患者健康状况、复查情况、用药情况和症状。
4. 内容：
   * 围绕冠脉CTA检查后、心脏相关疾病、检查和药物使用
   * 覆盖提供的对话片段
5. 语言特点：
   * 简洁、简短且直接
   * 非常的**口语化**，句子结构可以不完整，主谓宾等可以省略
   * 患者/家属的回答逻辑可以适当的混乱
   * 对话风格符合中文口语语境
   * 使用医学术语，但会适时解释
   * 循序渐进，从一般到具体
   * 灵活应变，根据患者回答调整

# 随访报告结果指南
## 1. 随访信息来源
- 定义：随访对象是否为本人
- 值域：["本人", "亲属"]
- 判定方法：综合对话内容，判断是否为本人接受随访

## 2. 受试者是否死亡
- 定义：受试者生命状态
- 值域：["是", "否"]
- 判定方法：
  - "是"：亲属明确提及受试者去世
  - "否"：其他所有情况

## 3. 受试者是否住院
- 定义：受试者是否接受住院治疗
- 值域：["是", "否", "未提及", "不确定"]
- 判定方法：
  - "是"：
    - 明确提及住院
    - 可推及住院的情况：支架手术、搭桥手术、瓣膜置换、脑血管造影DSA
  - "否"：
    - 明确否认住院
    - 提及未复查或仅健康体检
    - 仅提到造影但未明确是DSA还是CTA
    - 在社区医院就诊
  - "未提及"：无任何相关信息
  - "不确定"：受试者的逻辑混乱或自相矛盾，导致无法做出判断

## 4. 受试者是否手术
- 定义：受试者是否接受手术治疗
- 值域：["是", "否", "未提及", "不确定"]
- 判定方法：
  - "是"：
    - 明确提及手术
    - 明确的手术类型：支架、搭桥、瓣膜置换、脑血管造影DSA等
  - "否"：
    - 明确否认手术
    - 仅提及开药或保守治疗
  - "未提及"：无任何相关信息
  - "不确定"：受试者的逻辑混乱或自相矛盾，导致无法做出判断

## 5. 受试者是否用药
- 定义：受试者在随访期间是否使用药物
- 值域：["是", "否", "未提及", "不确定"]
- 判定方法：
  - "是"：随访期间使用药物（无论是否停药）
  - "否"：其他所有情况
  - "未提及"：无任何相关信息
  - "不确定"：受试者的逻辑混乱或自相矛盾，导致无法做出判断
